# Glean Maths

adds latex maths to [glean notes](https://glean.co/). powered by [katex](https://katex.org/)

## delimiting maths

| left  | right | display mode |
|-------|-------|--------------|
| $$    | $$    | yes          |
| \\[   | \\]   | yes          |
| \\(   | \\)   | no           |
